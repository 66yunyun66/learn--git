创建一个路径没有中文的文件夹  如learn--git   命令： mkdir learn--git
进入这个文件夹  命令：cd learn--git    查看当前位置  命令：pwd。
将这个文件夹设置为git可以管理的仓库  命令：git init。会在这个文件夹内部放置一些相应的git文件

将这个文件夹中的修改放进git仓库  分两步 第一：git add learn--git.txt. 添加是添加到暂存区
暂时将修改打上断点记录下来。
第二：将修改添加后提交 命令：git commit -m "这里面是对这次提交的说明"  提交是将修改从暂存区
提交给git让git将它管理记录下来。

add  一段时间内可多次使用，commit用一次就行。
名词解释  1：版本库：又叫仓库 ，可以理解为一个文件夹，里面的所有文件都可以被git管理，每次对
文件的修改，删除，git 都能跟踪，以便于将来还原。
